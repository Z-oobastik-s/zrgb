# RGB Minecraft - Text Generator 🎨

Мощный генератор RGB текста для Minecraft плагинов и чата с максимальной производительностью и современным дизайном.

**Автор:** Zoobastiks  
**Создан:** 2026 год

## ✨ Возможности

- 🎨 **Генерация RGB текста** с поддержкой множества форматов
- 🌈 **Палитра цветов** с выбором из стандартных Minecraft цветов
- 🎲 **Случайная генерация** из 1000+ вариантов цветов и градиентов
- 📋 **Автоматическое копирование** в буфер обмена одним кликом
- 🌍 **Многоязычность** (RU, UA, EN) с переключением на лету
- ⚡ **Максимальная производительность** с оптимизацией React 19
- 🎯 **Поддержка форматов**: & (Ampersand), § (Section), &x&r&r&g&g&b&b (Hex), MiniMessage
- 🎭 **Форматирование**: жирный, курсив, подчеркивание, зачеркивание, зашифрованный
- 🌈 **Rainbow режим** для создания радужного градиента
- 🎨 **Кастомный RGB пикер** для точного выбора цвета

## 🚀 Технологии

- **Next.js 15** - React фреймворк нового поколения
- **React 19** - Последняя версия React
- **TypeScript** - Типобезопасность
- **Tailwind CSS** - Современные стили
- **Framer Motion** - Плавные анимации
- **next-intl** - Интернационализация

## 🎯 Использование

1. Введите текст в поле ввода
2. Выберите формат кода (Ampersand, Section, Hex, MiniMessage)
3. Настройте форматирование (жирный, курсив и т.д.)
4. Выберите цвет из палитры или используйте случайный
5. Скопируйте результат одним кликом
